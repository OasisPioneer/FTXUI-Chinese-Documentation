‍[!WARNING] 此功能仍在开发中，API 在未来版本中可能会发生变化。 您的贡献对于帮助我们改进 FTXUI 中 C++20 模块的兼容性和可用性至关重要。如果您遇到任何问题或有任何建议，请提出问题。

FTXUI 实验性地支持 C++20 模块，以减少编译时间并改进代码组织。每个头文件都有一个对应的模块。

CMake 和 Ninja 示例

import ftxui;
int main() {
auto screen = ftxui::ScreenInteractive::TerminalOutput();
auto button = ftxui::Button("Click me", screen.QuitClosure());
screen.Loop(button);
return 0;
}
